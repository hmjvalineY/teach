⚙️ 系統部署提示 (Giscus 設定)

目前 Giscus 留言板需要您在專案倉儲中啟用 Discussions 功能，並安裝 Giscus GitHub App。
請自行替換 discussions.html 下方的 data-repo、data-repo-id 與 data-category-id 為您真實的倉儲數值即可正式啟用此頁面功能。
