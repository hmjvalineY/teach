賦能高中教育：Google AI 工具應用指南

Google Stitch 與 Google Jules 在六大領域的教學與學習實踐

📊 核心工具快速比較

這兩款工具能將「想點子」到「做成品」的距離大幅縮短，是實踐跨領域專題與教師數位賦能的最佳利器。

工具名稱 核心功能 使用門檻 高中校園最佳應用場景
Google Stitch
(視覺設計師)		 透過文字或語音，自動生成網頁 / App 介面與前端程式碼。 極低
適合全校師生跨領域使用		 跨科專題網頁展示、數位互動教材介面、高質感學習歷程檔案。
Google Jules
(程式碼代理人)		 綁定 GitHub，在背景自動閱讀程式碼、抓 Bug、寫測試腳本。 較高
需具備基礎程式碼與版本控制概念		 資訊科技科展、進階程式專題、教師開發自動化批改腳本。

🏫 六大領域應用實例

📜 國文科 (Chinese)

核心：文學數位具象化與文本自動化

  • Stitch 古典地圖網頁：學習貶謫史時，生成帶有互動地圖與詩句對照的網頁。
  • Stitch 孔子人格測驗：設計模擬與論語人物對話的心理測驗 App 原型。
  • Jules 成語排版器：教師寫 Python 抓每日成語，Jules 修復 Bug 並自動排版講義。
  • Jules 情感分析工具：教師開發古文情緒分析程式，Jules 優化分析邏輯。
  • 整合 桃花源記文字遊戲：學生改編課文，Stitch 做復古介面，Jules 寫分支邏輯。

🗣️ 英文科 (English)

核心：情境互動與自動化語言輔助

  • Stitch 外籍生導覽 App：學生寫全英文文案，生成校園地圖與 Q&A 介面。
  • Stitch 單字情境解謎：教師製作虛擬密室網頁，輸入正確單字/文法解鎖。
  • Stitch 學習歷程儀表板：學生設計個人介面，記錄多益進度與英語配音作品。
  • Jules 新聞摘要腳本：教師設腳本抓 BBC 新聞，Jules 寫測試確保網頁改版不失效。
  • Jules 單字推播工具：學生寫定期發送單字的手機推播程式，Jules 解決 API 串接錯誤。

📐 數學科 (Mathematics)

核心：抽象邏輯視覺化與運算模擬

  • Stitch 函數動態模擬器：教師設計網頁介面，拖動滑桿觀察三角函數圖像變化。
  • Stitch 理財複利工具：學生做學貸試算或儲蓄目標 App，展現生活數學應用。
  • Stitch 統計儀表板：學生校內問卷調查後，生成即時圓餅圖與長條圖網頁。
  • Jules 蒙地卡羅算 π：學生寫程式跑數萬次模擬，Jules 優化迴圈效能。
  • Jules 題庫自動生成：教師寫腳本隨機出題，Jules 檢驗邏輯避免生出無解題目。

🌍 社會科 (Social Studies)

核心：公民參與與空間資訊展示

  • Stitch 虛擬連署平台：公民課發起校園議題，做出專業連署網頁與回饋介面。
  • Stitch 地方創生地圖：地理課將家鄉特色店家製作成互動式導覽網頁地圖。
  • Stitch 歷史 3D 時間軸：教師生成橫向滑動時間軸，直觀呈現東西方歷史共時性。
  • Jules Open Data 爬蟲：教師抓取政府數據，Jules 背景修復因網站更新導致的失效。
  • Jules 選戰聲量監測：模擬選舉中寫程式分析社群聲量，Jules 協助清洗大量數據。

🔬 自然科 (Science)

核心：實驗數據獲取與虛擬實驗室

  • Stitch 虛擬實驗儀表板：學生製作滴定實驗介面，模擬化學試劑混合變色反應。
  • Stitch 互動式週期表：教師製作點擊元素即跳出 3D 結構圖與實驗影片的網頁。
  • Stitch 星象觀測日誌：地科觀星後，學生將數據填入網頁產出個人精美日誌。
  • Jules 感測器除錯：物聯網專題中，Jules 協助找出 Arduino 感測器斷線的 C++ Bug。
  • Jules 物種動態模擬：教師開發捕食者消長程式，Jules 撰寫變數測試讓模擬更準確。

🎨 藝能科 (Arts/Vocational)

核心：作品集數位化與生成式藝術

  • Stitch 線上 3D 藝廊：美術班學生生成個人網頁，用瀑布流展示高中三年創作。
  • Stitch 卡路里追蹤 App：家政課專題，設計管理校園午餐營養的介面原型。
  • Stitch 樂理互動平台：音樂教師製作虛擬琴鍵，點擊發聲並顯示對應五線譜。
  • 整合 體育即時計分板：舉辦比賽時，Stitch 生成前端顯示，Jules 處理後端計分同步。
  • Jules 生成式藝術：學生嘗試 p5.js 寫藝術特效，Jules 解決語法錯誤確保網頁順利運行。

💡 導入建議： 一般師生可從 Stitch 入手，將報告升級為互動網頁；資訊社群或科展團隊則推薦使用 Jules 挑戰複雜開發計畫。
