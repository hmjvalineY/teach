🛠 直接在雲端末端展示運算結果

Jupyter Notebook 廣受 AI 工程師與資料科學家的喜愛，但在本機端通常需要架設 Anaconda 等笨重的環境。只要上傳到 GitHub，一切都變得簡單直觀。

支援渲染的內容：

  • Python 程式碼區塊：帶有標準的語法高亮顯示。
  • 數學公式：完美支援 LaTeX 數學符號引擎 (MathJax)。
  • 視覺化圖表：由 Matplotlib 或 Seaborn 在本機產生的長條圖、散佈圖、熱力圖等。

你只需要將 .ipynb 檔案上推到存放庫，GitHub 會在背後自動將其轉換為靜態唯讀網頁，方便分享給教授或同學檢閱。
