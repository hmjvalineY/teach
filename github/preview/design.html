圖文與設計原檔預覽

🛠 預覽 Photoshop 與 PDF 免開軟體

如果團隊中缺乏設計軟體，那 GitHub 也能作為你的看圖平台。

1. PSD 原檔預覽 (.psd)

上傳 Photoshop 專案檔 (.psd) 後，GitHub 能夠幫你渲染出最終的點陣圖畫面。

超猛應用：Swipe 比對工具

當設計師更新了 PSD 檔（上傳新的版本 Commit），你可以將檢視模式切換至 "Swipe"。你會看到畫面上有一條控制桿，左右滑動即可如拉動布簾般，對照「修改前」與「修改後」在像素等級的微妙變更。

(若您有 PSD 檔案，可以試著上傳到這個專案看看效果！)

2. PDF 與 SVG

GitHub 同樣也提供這兩者的原生檢視。在提交 SVG 向量檔案時，你甚至可以直接用源碼編輯，網頁介面會即時展現路徑變更的結果。

🎯 實作任務與示範：

我已經在這個資料夾準備了一個 example_logo.svg，也在上層資料夾有一份 .pdf 簡報，您可以上傳它們，然後直接在 GitHub 上預覽：

  1. 透過 GitHub Desktop 將 example_logo.svg 檔案上傳 (Push) 至您的 Repo。
  2. 來到 GitHub 網站點開檔案，即可免裝軟體看到設計好的向量圖或 PDF 文件！
