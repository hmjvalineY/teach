第二大腦跨平台免費同步

🛠 擺脫雲端儲存空間的高昂訂閱費

如果你是一位狂熱的筆記記錄者，可能會使用 Notion、Evernote 或者是最近極度火紅的 Obsidian 與 Logseq 來構建個人知識庫（第二大腦）。如果透過它們內建的官方端到端加密同步功能，往往需要付不小的月費。

私有倉庫 (Private Repo) 的威力

由於 Obsidian 與 Logseq 這些新世代軟體，底層其實就是一般的 Markdown 實體純文字檔 的資料夾集合。因此：

  1. 在 GitHub 開啟一個 Private（不公開）專案。
  2. 在軟體內下載社群開發的插件：如 Obsidian Git。
  3. 插件會自動把你的筆記資料夾當作一個 Git 專案處理。
  4. 全自動背景同步：每隔 10 分鐘，或者是當你開啟/關閉軟體時，插件會安靜地在底層自動幫你 Commit 和 Push 筆記到雲端。

不只能讓多台電腦和手機保持無縫同步，你還獲得了「筆記歷史修改紀錄」，如果不小心把你兩週前寫的文章刪掉了，你可以像工程師回復程式碼一樣，把文章完美倒轉回來。
