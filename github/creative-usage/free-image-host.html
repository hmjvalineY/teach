🛠 截取微軟的超級 CDN 資源

架設個人部落格或是在外面的論壇（如巴哈姆特、PTT 網頁版）發文時，圖片空間常是個問題（Imgur 可能會刪圖，自己買機器則要算流量費）。這時 GitHub Issue 就是你最堅強的後盾。

操作四步驟：

  1. 進入自己任意一個 GitHub 專案，點擊上方的 Issues，然後按 New Issue 開啟新的發言框。
  2. 使用滑鼠將你的圖片檔（PNG, JPG 或 GIF）直接拖曳進入該文字框；或是使用截圖工具後，直接在此框內按下 Ctrl+V。
  3. 等待兩秒鐘上傳，文字框內便會神奇地產生一段包含實體連結的 Markdown 語法，長得像這樣：
    ![螢幕擷取畫面](https://github.com/user-attachments/assets/abc123def...)
  4. 千萬不要按送出 Issue 按鈕！ 你只要偷偷把括號內那串網址複製下來，直接拿去你的部落格或任何論壇貼上即可，該網址由全球最快的網路節點分發支援，且永久有效無限容量！
