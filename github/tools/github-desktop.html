專為非工程師設計的 GitHub Desktop

🛠 拖拖拉拉搞定版本控制

一想到 Git，多數人腦袋就會浮現一整片黑底白字的終端機與密密麻麻的 git commit, git push, git fetch 咒語，這讓企劃人員和美術望之卻步。

官方解決方案：GitHub Desktop

微軟官方專門為不想背誦指令的用戶開發了一套完全圖形化 (GUI) 的客戶端軟體。你可以在 desktop.github.com 免費下載。

  • 複製專案 (Clone)： 專案列表點及一下即可把幾百 MB 的資料夾下載到桌面。
  • 觀察變更： 將新畫好的圖檔丟進資料夾後，軟體左側會亮起綠燈，提醒你有新檔案加入。
  • 存檔與上傳 (Commit & Push)： 只要在左下角輸入一個便條標題「更新首圖設計」，按下藍色大按鈕，再點擊右上方的 Push，就完成了史詩級的伺服器同步工作！這一切都不需要呼叫出小黑窗。
