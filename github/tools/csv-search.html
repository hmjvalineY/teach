全球數據尋寶 (搜尋副檔名)

🛠 取得真實世界資料集最快的方法

在教授地理或是資工的演算法分析時，總需要龐大又真實的 Dataset 供學生練習。相對於去各國政府開放資料網慢慢摸索，GitHub 是全球最密集的資料集寶庫。

搜尋秘技：利用副檔名限制

進入首頁上方搜尋框，你可以下達精確的過濾條件：

  • 輸入 covid statistics extension:csv
    你可以瞬間挖出世界各國研究員正在使用與整理好的通用 Excel 表格，下載下來馬上就能在課堂上供統計分析使用。
  • 輸入 pokemon extension:json
    想讓學生寫網頁串 API 發 App，這招能幫你找到有人手刻好世界上所有寶可夢圖像與數值屬性的純文字結構檔 (JSON)。
