禪意語錄 (官方隱藏彩蛋)

🛠 開發者的哲學

這個世界最大的程式碼交換所，其工程師心中都有著一套極簡的哲學。

如何呼叫禪意大師

開啟新分頁，在瀏覽器的網址列輸入這串特殊的 API 入口點：

https://api.github.com/zen

每次重整畫面，都會跳出不同的一句名言警句（純文字顯示在畫面左上角）：

  • "Anything added dilutes everything else." (任何增加的事物都在稀釋原有的核心)
  • "Mind your words, they are important." (注意你的言詞，它們很重要)
  • "Speak like a human." (說人話)
  • "Non-blocking is better than blocking." (流暢不阻塞總是優於卡頓)

這是一個有趣的網路空間小彩蛋，很適合在需要沉澱思緒時玩弄一番。
