👩‍🏫 教師教學應用

管理大型協作探究專案

公民老師帶領「在地文化保存」專題，利用 GitHub 的 Issues 功能系統化管理各組的任務與進度。

📋 Issue 任務看板模擬

📌 待辦
📷 實地訪談記錄
📊 問卷設計
🔄 進行中
📚 文獻蒐集
🗺️ 地圖標記
✅ 已完成
📝 研究動機撰寫

🛠 操作方式

  1. 老師在 GitHub 上建立專案 Repository
  2. 利用 Issues 功能為每項任務建立獨立的議題
  3. 為每組學生指派不同的 Issue（如：實地訪談、文獻蒐集、問卷設計）
  4. 學生在對應的 Issue 下方留言回報進度、上傳參考資料
  5. 老師隨時掌握各組進度，並在 Issue 中給予指導回饋
👨‍🎓 學生學習應用

利用 GitHub Pages 建立數位化策展

地理課的 GIS（地理資訊系統）專題，學生將社區歷史地標轉化為互動式導覽網站。

🗺️ 互動地圖網站建置步驟

  1. 蒐集社區歷史地標的經緯度資料，整理為 GeoJSON 格式
  2. 結合簡單的 HTML 與開源地圖工具（如 Leaflet.js）
  3. 撰寫每個地點的歷史故事介紹文字
  4. 利用 GitHub Pages 功能，將 Repository 一鍵部署為免費網站
  5. 成品為「在地歷史文化導覽互動地圖」網站，作為期末成果發表

💡 成果價值：不僅是課堂報告，更是一個能實際分享給社區居民使用的數位工具。
