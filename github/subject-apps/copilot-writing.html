🤖 GitHub Copilot 非程式碼寫作應用

在 VS Code 中開啟 Markdown (.md) 檔案，Copilot 就能化身為超級寫作助手

👩‍🏫 教師應用

快速生成教案大綱與測驗題

英文老師在 Markdown 檔案中輸入標題，Copilot 會根據上下文自動推敲並生成完整的教學素材草稿。

📝 實際操作範例

  1. 在 VS Code 中建立一個新的 .md 檔案
  2. 輸入標題 ## 關於全球暖化的英文閱讀測驗設計
  3. 按下 Enter 換行後，Copilot 會自動預測並生成內容
  4. 生成內容可能包含：單字表、閱讀文章、A/B/C/D 選擇題
  5. 只需按下 Tab 鍵接受建議，再進行微調即可

💡 省時效果：原本需要 1-2 小時的備課工作，可以縮短到 15 分鐘完成初稿。

👨‍🎓 學生應用

跨語系資料整理與寫作引導

學生在進行自主學習或專題研究時，可以利用 Copilot 協助處理多語言資料與克服寫作瓶頸。

📝 實際操作範例

  1. 學生在進行「自主學習：探討日本高齡化社會」研究
  2. 將找到的日文資料貼入編輯器中
  3. 加上註解 <!-- 請將上述資料整理成三個中文重點 -->
  4. Copilot 會自動生成中文摘要
  5. 撰寫報告時，只要打出「第一章：」，Copilot 就會順著語境推敲出後續段落

💡 關鍵優勢：幫助打破「寫作瓶頸 (Writer's Block)」，讓思路不再卡住。

🏫 行政 / 社團

自動生成企劃書與公文模板

學生會幹部在撰寫各類企劃書時，Copilot 可以自動補齊標準化格式與建議文案。

📝 實際操作範例

  1. 建立新的 Markdown 檔案
  2. 輸入 # 迎新營隊贊助企劃書
  3. Copilot 會自動補齊以下內容：
    • 活動宗旨 — 說明活動目的與意義
    • 預期效益 — 量化的活動成果預估
    • 贊助回饋方案 — 給贊助商的回饋計畫
    • 預算表 — 分項列出的經費規劃
