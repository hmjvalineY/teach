引用此作品與抖內機制

🛠 CITATION.cff - 讓開源獲得學術認可

如果是教學或學術軟體，在專案根目錄放一個名為 CITATION.cff 的檔案，你的專案側邊欄就會多出一顆 "Cite this repository" 的按鈕，讓其他人可以一鍵複製 APA 等引用格式。

撰寫範例：

cff-version: 1.2.0
message: "If you use this software, please cite it as below."
authors:
  - family-names: "Lin"
    given-names: "Teacher"
title: "Interactive Teaching Portal"
version: 1.0.0
date-released: 2025-01-01

🛠 FUNDING.yml - 開啟抖內小費箱

在專案建立 .github/FUNDING.yml，輸入你的贊助網址：

# 支援的贊助平台
patreon: my_username
ko_fi: my_username
custom: ['https://example.com/donate']

這會在專案右上方亮起一個顯眼的 "💗 Sponsor" 按鈕！
