信箱懶人回覆留言法

🛠 無須開啟網頁的超速溝通術

對於負責管理多個專案的老師或專案經理，每天可能收到大量的 Issue 提問或是 PR 的通知。如果每一個都要點開網頁、登入、送出留言，那實在太耗時了。

如何使用：

  1. 在 GitHub 的 Settings -> Notifications 中開啟接收 Email 通知。
  2. 當有人在討論串標記 @你的帳號 或發布新回覆時，你的 Gmail 信箱夾就會收到信。
  3. 重點來了：你可以直接在手機或電腦上點選「回覆該封 Email信件」。
  4. 打上純文字訊息（如：「這部分已了解，下週修正」）後送出 Email。
  5. GitHub 伺服器會解析這封回信，並自動使用你的帳號，把那段文字轉為對應 Issue 的實時網頁留言！

註：這種回覆方式會自動捨棄你 Email 預設的簽名檔等無關資訊，非常聰明乾淨。
