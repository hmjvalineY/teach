自動存檔核取方塊 (Task Lists)

🛠 網頁上的互動魔法

這是一個非常低調卻極致好用的 Markdown 擴展功能。

一般寫法：

- [ ] 這是我還沒完成的工作
- [x] 這是已經查閱完畢的資料

當這段語意在 Issue 討論串，或甚至是普通的 README 文件中顯示時，它不僅僅只是純文字圖示，它會渲染成真實可點擊的核取方塊。

💡 神奇之處

不論是誰（只要有編輯權限），直接在閱讀畫面上用滑鼠「點擊」那個空方塊。方塊會立刻打勾，並且 GitHub 伺服器會在背後自動發起一次 Commit 修改並儲存該檔案，完全不需要你手動點開 "Edit file"！適合用來當作團隊共同確認事項進度的 Check list。
