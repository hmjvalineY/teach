一秒變編輯器（按下 .）

🛠 實際操作示範

在瀏覽程式碼時，不需要安裝任何軟體，直接在網頁上敲擊鍵盤的 `.` (句號) 鍵。

  1. 進入任何一個你想查看的 GitHub 專案首頁。
  2. 確認輸入法在英文狀態，按下 . 鍵。
  3. 網址會自動從 github.com/... 秒換為 github.dev/...。
  4. 畫面將載入與你電腦上一模一樣的 VS Code 雲端編輯器！

💡 實用情境

當你想深入追蹤某個函數的定義與來龍去脈，如果只在一般 GitHub 網頁版點擊連結會非常慢且容易迷路。開啟 VS Code 視圖後，你就能使用熟悉的「全域搜尋」與「檔案樹結構」，大幅提昇閱讀開源代碼的效率。
