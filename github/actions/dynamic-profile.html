動態個人履歷 (Profile README)

🛠 讓你的名片活起來

在 GitHub 建立一個與自己帳號同名的特殊 Repository (例如你的帳號是 alice，你就建一個名叫 alice 的專案)，裡面的 README.md 就會變成你個人網頁首頁的介紹版面。

利用 Actions 將靜態文字動態化：

許多開發者利用 GitHub Actions 的定時排程，每天去抓取並更新這個首頁的文字：

  • 聽歌紀錄：使用網路上開源的 Spotify Action 插件，每天抓取你最近聽了什麼歌，並把專輯封面貼在首頁。
  • 部落格最新文章：利用 RSS Reader Action，當你在 Medium 或是個人部落格發布新文章時，GitHub 首頁的清單會自動增加一條超連結。
  • 雷達圖與貢獻指數：加上 github-readme-stats 標籤，就能在首頁展開酷炫的雷達圖紙。
