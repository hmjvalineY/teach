免費網站保全 (自動 Ping 與警報)

🛠 不花錢的 Uptime Monitor

如果自己或學校架設了網站，最怕的就是半夜當機而無人知曉。我們可以使用 Actions 每 15 分鐘去戳一次你的網站，掛掉時發送 LINE 或 Email 警報。

如何實作網站存活檢查

name: 網站自動保全系統
on:
  schedule:
    - cron: '*/15 * * * *' # 每 15 分鐘執行一次

jobs:
  check_site:
    runs-on: ubuntu-latest
    steps:
      - name: 檢查我的學校網站
        uses: jcmartins/uptime-monitor@v1
        with:
          url: "https://my-school-website.edu"
          timeout: 5000
          
      - name: 網站死掉時發出警告
        if: failure()
        run: |
          # 呼叫你設定好的 LINE Notify Webhook
          curl -X POST -H "Authorization: Bearer ${{ secrets.LINE_TOKEN }}" \
               -F "message=警報：伺服器沒有回應了！" \
               https://notify-api.line.me/api/notify
